概述
---
* `async` 一个用于解决callback hell（回调嵌套）的类库。
* 类似的库挺多，例如angularjs中的Q.js，但目前async关注率较高，这一次我就随(ying)大(yu)众(lan)吧！：D
* async主要实现了三个部分的流程控制功能
	* 集合　　：Collections
	* 流程控制：Control Flow
	* 工具类　：Utils
* 上一条具体介绍可以查看 [粉丝日志](http://blog.fens.me/nodejs-async/) 内的文章，中文注释写很详细。
